Judo +100kg

Introduction aux Jeux Olympiques

Le judo, Sport de combat et discipline olympique, a fait son entrée aux Jeux Olympiques en 1860. Les athlètes de plus de 100kg, par leur force et leur technique, offrent des combats spectaculaires.

Présentation de l'épreuve

L'épreuve de judo des plus de 100kg est une catégorie reine du judo. Les judokas de cette catégorie sont connus pour leur puissance et leur envergure. Ils s'affrontent dans des combats souvent très physiques où chaque technique est décisive.

Où se déroulent les compétitions ?

Les compétitions de judo des Jeux Olympiques de Paris 2024 se dérouleront à l'Arena Champ-de-Mars, au cœur de Paris. Un lieu emblématique offrant une expérience inoubliable.

Règles du jeu

Le judo est un sport basé sur le respect et la maîtrise de soi. Les principales règles sont :

  • L'ippon: C'est la victoire directe. Il peut être obtenu par une projection, une immobilisation, une strangulation ou une clé articulaire.
  • Le waza-ari:C'est une demi-victoire. Il est attribué pour une technique moins efficace qu'un ippon.
  • Le shido:C'est une pénalité. Un judoka qui commet une faute reçoit un shido. Trois shidos entraînent la disqualification.

Les phases de l'épreuve

  • Les poules: Les judokas sont répartis en poules et s'affrontent pour se qualifier pour les phases finales.
  • Les tableaux: Les meilleurs judokas de chaque poule se qualifient pour les tableaux, où ils s'affrontent en élimination directe.
  • Les finales: Les deux finalistes s'affrontent pour le titre olympique.
